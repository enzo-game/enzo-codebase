## 規格（每份 MD 開頭也有）
- **檔名**：每張存成 `public/images/cards/<id>.jpg`。id 見 MD 的「檔名」欄，例如 `gen-0201.jpg`、`leg-l40.jpg`。**檔名一定要跟 id 一致**，否則工程端登錄不到。
- **風格**：厚塗寫實／水彩感的台灣山林插畫，5:7 直式，構圖飽滿，無文字、無邊框、無浮水印。
- **文化框限**：不畫人形、不畫神聖器物或祭儀場景、不畫百步蛇。傳說卡（`leg-*`）另有個別提示，一律謹守地景／自然、不畫人形。
- **id 穩定**：批次順序與 id 固定，不會變動 → **批次 1 已生的 194 張不需重做**。
